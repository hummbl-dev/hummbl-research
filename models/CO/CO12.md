# CO12 - Event-Driven Composition

> Compose systems through asynchronous event messaging.

## Description

Event-Driven Composition uses events to decouple components, enabling reactive and scalable architectures.

## Example

E-commerce: "Order placed" event triggers inventory update, email, and shipping—components compose without direct calls.
