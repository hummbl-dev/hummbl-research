# CO15 - Composite Patterns

> Treat individual and group objects uniformly.

## Description

Composite Patterns compose hierarchies where leaves and branches share interfaces, simplifying complex structures.

## Example

File system: Files and folders both have "size" method. Compose directory trees uniformly.
