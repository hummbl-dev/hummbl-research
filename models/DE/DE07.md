# DE07 - Cause-Effect Diagrams

> Break down effects into contributing causes using fishbone diagrams.

## Description

Cause-Effect Diagrams (Ishikawa diagrams) decompose problems into categories of causes (e.g., people, process, equipment), systematically exploring root factors.

## Example

For "late deliveries": Causes include "supplier delays" (materials), "traffic" (environment), "poor planning" (process). This guides targeted fixes.
