# DE12 - Critical Path Analysis

> Identify the sequence of tasks that determine project duration.

## Description

Critical Path Analysis decomposes projects into tasks with dependencies, highlighting the longest path for scheduling.

## Example

Construction: Foundation → Framing → Roofing. Delays in framing (critical) extend the whole project.
