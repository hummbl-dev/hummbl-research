# DE13 - Functional Analysis

> Break systems into functions and analyze their performance.

## Description

Functional Analysis decomposes systems by functions (what they do), evaluating efficiency and interactions.

## Example

Car engine: Functions like "ignite fuel," "generate power." Analyze each for optimization.
