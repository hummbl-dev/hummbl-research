# P11 - Cognitive Biases Awareness

> Recognize and counteract mental shortcuts that distort thinking.

## Description

Cognitive Biases Awareness reframes decisions by identifying common biases (e.g., confirmation bias, anchoring), allowing conscious correction for more objective analysis.

## Example

In hiring: Bias towards "similar-to-me" candidates. Awareness prompts diverse shortlists, improving team balance.
