# P15 - Cultural Context

> Consider cultural norms and values in problem-solving.

## Description

Cultural Context reframes issues by accounting for diverse backgrounds, avoiding ethnocentric biases and fostering inclusive solutions.

## Example

Global product: US focus on individualism vs. collectivist cultures. Adapt features for cultural fit, like group-sharing options.
