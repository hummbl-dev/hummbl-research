# RE04 - Adaptive Planning

> Plan with flexibility, adjusting based on new information.

## Description

Adaptive Planning uses recursion to update strategies as conditions change. It embraces uncertainty by building in review points, making plans resilient to surprises.

## Example

In agile development, plan sprints but adapt backlog based on stakeholder feedback. Recursion ensures alignment with evolving goals.
