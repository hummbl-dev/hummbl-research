# RE11 - Bootstrapping

> Build upon initial resources through iterative self-improvement.

## Description

Bootstrapping uses recursion to start small and compound gains, leveraging outputs as inputs for growth.

## Example

Startup: Use initial sales to fund marketing, which drives more sales. Recursively scale without external capital.
