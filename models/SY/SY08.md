# SY08 - Model Fusion

> Merge multiple models into a unified, more powerful one.

## Description

Model Fusion synthesizes disparate models (e.g., economic + psychological) into hybrids, capturing combined strengths for better predictions.

## Example

Behavioral economics: Fuse rational choice (economics) with cognitive biases (psychology) for realistic decision models.
