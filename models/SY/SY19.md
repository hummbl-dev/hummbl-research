# SY19 - Meta-Model Selection

> Automatically recommends a sequence of HUMMBL models based on a problem description.

## Description

SY19 is a meta-model that recommends which HUMMBL models to use for a given problem. It uses the relationship graph (333 relationships across 120 models) to walk from primary models and suggest relevant models based on:

- **Relationship strength** (0.0-1.0)
- **Relationship type** (SCAFFOLDS, COMPOSES_WITH, REFINES, etc.)
- **Model centrality** (how connected a model is)
- **Graph distance** (models closer to primaries score higher)

The scoring formula:
```
score = strength Ã— type_weight Ã— direction_weight Ã— centrality_weight Ã— hop_decay
```

Type weights:
- SCAFFOLDS: 1.0
- COMPOSES_WITH: 0.9
- REFINES: 0.8
- PARALLELS: 0.5
- CONTRASTS_WITH: 0.4
- CONFLICTS: 0.3

## Implementation

**Tool:** `tools/sy19_recommend.py`

**Usage:**
```bash
# With explicit primaries
python tools/sy19_recommend.py "problem description" --primaries DE07 DE06

# Auto-detect primaries from keywords
python tools/sy19_recommend.py "multi-service system with bottlenecks"

# Save to file
python tools/sy19_recommend.py "problem" --output recommendations.md
```

**Features:**
- Automatic primary model detection from keywords
- Graph traversal with configurable depth (max_hops)
- Centrality-weighted scoring
- Primary model boost (ensures primaries stay in top-K)
- Detailed reasoning (why each model was recommended)
