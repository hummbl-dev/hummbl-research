# IN09 - Red Teaming

> Simulate adversarial attacks to test defenses.

## Description

Red Teaming inverts defensive thinking by role-playing as an attacker to exploit weaknesses. It uncovers vulnerabilities in systems, plans, or ideas before real threats arise.

## Example

In cybersecurity, a red team attempts to breach defenses. Findings lead to patches, like fixing a weak password policy.
